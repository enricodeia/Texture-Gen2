How to Use Texture Gen v2

1. Uploading a Texture

Start by uploading a base texture image using the drop area or Browse Files button. The application supports JPG, PNG, and WebP formats.

2. Texture Maps Generation

Once uploaded, the app automatically generates the following PBR texture maps:

  • Base Color Map: The original texture with adjustable intensity
  • Normal Map: Creates surface detail without geometry
  • Roughness Map: Controls how rough or smooth the surface appears
  • Displacement Map: Creates actual geometric displacement
  • Ambient Occlusion Map: Adds shadowing in crevices

3. Adjusting Settings

Use the sliders to fine-tune each texture map:

  • Adjust strength values for each map type
  • Change metalness to make surfaces appear more metallic
  • Modify light position to see how lighting affects your textures

4. Texture Mapping Controls

Perfect how your texture wraps on the 3D preview:

  • Use Repeat X/Y to control tiling frequency
  • Adjust Offset X/Y to position the texture precisely
  • These settings affect all generated maps uniformly

5. 3D Preview Controls

Inspect your texture from all angles:

  • Use the X and Y rotation sliders to rotate the sphere
  • Toggle auto-rotation with the rotation button

6. Exporting

Export your texture maps in several ways:

  • Download individual maps using their download buttons
  • Export all maps together as a ZIP file
  • Get ready-to-use Three.js code that implements your material

About Texture Gen v2

Version

Texture Gen v2.0 - Released March 2025

Description

Texture Gen is a powerful web-based PBR texture generator. It enables you to create professional texture map sets from a single source image. The application uses advanced image processing algorithms to generate normal, roughness, displacement, and ambient occlusion maps that deliver realistic materials for 3D rendering.

Features

  • Instant PBR texture map generation
  • Real-time 3D preview
  • Customizable texture mapping settings
  • Three.js code export
  • Batch export as ZIP

Technologies

Built with HTML5, CSS3, JavaScript, and Three.js
